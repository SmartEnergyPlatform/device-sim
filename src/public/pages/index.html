Sensor Content

the system provides some convenience functions for use in the content code

Graph

  • graph(id).f(x)
  • graph(id).interval(min, max).f(x)
  • graph(id).secInMin()
  • graph(id).secInHour()
  • graph(id).secInDay()
  • graph(id).minInHour()
  • graph(id).minInDay()
  • graph(id).hourInDay()

code completion helps to find existing graph ids

Example: graph("329ccbe2-1cd5-2172-604d-d1063cbf6bfb").interval(1, 9).f(count)

Random

random(min, max, decimalPlaces)

Sensor Format

code completion helps to find a placeholder for timestamps, thouse placeholders will be filled by the system and not the user

  • caption: "TIME_STAMP (e.g. 1478014571168)"

    value: "{{{TIME_STAMP}}}"

  • caption: "TIME_UTC (e.g. Tue, 01 Nov 2016 15:43:09 GMT)"

    value: "{{{TIME_UTC}}}"

  • caption: "TIME_ISO (e.g. 2016-11-01T15:39:15.378Z)"

    value: "{{{TIME_ISO}}}"

  • caption: "TIME_STRING (e.g. Tue Nov 01 2016 16:35:56 GMT+0100 (CET))"

    value: "{{{TIME_STRING}}}"

  • caption: "TIME_DATE (e.g. Tue Nov 01 2016)"

    value: "{{{TIME_DATE}}}"

  • caption: "TIME_TIME (e.g. 16:41:43 GMT+0100 (CET))"

    value: "{{{TIME_TIME}}}"
